# نظام التسعير الذكي - مهووس v26.0

## 📋 نظرة عامة

تطبيق ويب متطور مبني باستخدام **Streamlit** و **Python** لأتمتة ومساعدة فريق التسعير في تحليل أسعار المنافسين واتخاذ قرارات تسعير ذكية لمتجر عطور إلكتروني.

## ✨ الميزات الرئيسية

- **محرك مطابقة ذكي:** يستخدم `rapidfuzz` و Gemini AI للمطابقة الدقيقة بين المنتجات
- **تحليل أسعار المنافسين:** بحث تلقائي عن أسعار المنافسين وتحليل الفروقات
- **قواعد تسعير قابلة للتخصيص:** نظام أتمتة ذكي لاتخاذ قرارات التسعير
- **تكامل مع Make.com:** إرسال تلقائي للقرارات إلى أنظمة خارجية
- **إدارة قاعدة بيانات:** تخزين شامل للبيانات والسجلات التاريخية
- **واجهة مستخدم تفاعلية:** عرض بصري واضح للنتائج والقرارات

## 🚀 البدء السريع

### المتطلبات

- Python 3.8 أو أحدث
- pip (مدير الحزم)

### التثبيت

1. استنساخ المستودع:
```bash
git clone https://github.com/mahwoussa-boop/vision-2030-v26.git
cd vision-2030-v26
```

2. تثبيت التبعيات:
```bash
pip install -r requirements.txt
```

3. تعيين متغيرات البيئة:
```bash
export GEMINI_API_KEY="your_gemini_api_key"
export WEBHOOK_UPDATE_PRICES="your_make_webhook_url"
export WEBHOOK_NEW_PRODUCTS="your_make_webhook_url"
```

4. تشغيل التطبيق:
```bash
streamlit run app.py
```

## 🔧 الإعدادات

يتم تخزين جميع الإعدادات في `config.py`. يمكنك تخصيص:

- **إعدادات المطابقة:** `MATCH_THRESHOLD`, `HIGH_CONFIDENCE`, `REVIEW_THRESHOLD`
- **الألوان:** `COLORS`
- **قواعس الأتمتة:** `AUTOMATION_RULES_DEFAULT`
- **العلامات التجارية:** `KNOWN_BRANDS`

## 📊 الاستخدام

1. **رفع الملفات:** قم برفع ملفات Excel/CSV تحتوي على منتجاتك وأسعار المنافسين
2. **التحليل:** سيقوم التطبيق بتحليل البيانات وإيجاد المطابقات
3. **المراجعة:** راجع النتائج واتخذ القرارات المناسبة
4. **الإرسال:** أرسل القرارات إلى Make.com لتحديث الأسعار تلقائياً
